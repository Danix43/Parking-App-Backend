# Parking App Backend

## Description

The backbone of the a parking app. Rest API that serves data about parking spots, prices and current occupation.
!Note: This is only a demo and doesn't work in real life. There are more well made applications made for that.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
